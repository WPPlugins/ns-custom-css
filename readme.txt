This plugin help to write easly custom css with no code required! Works with a lot of themes!

== Description ==
This plugin allow you to add your custom css code without edit your theme.
Page option will appear in your WordPress backend menu, and here you can add your css.
You only have to digit your class or id and save it!
The snippet code will added in your header, in a very clean and simple mode.

PREMIUM FEATURES of custom css:

You can **add different custom css** code **for any page**.
You can also add different custom css code in **article, archive and category pages**

== Installation ==

1. Upload the entire 'ns-custom-css' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Custom Css' and digit your code in the textarea
